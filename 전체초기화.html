온라인 Firestore 전체 초기화 도구

이 도구는 보드만 초기화하지 않고, 온라인 Firestore에 저장된 커리큘럼/과목카드/학급/학생/수강명단/교사/교실/시간표 데이터를 모두 삭제합니다.

실행 전 반드시 시간표 화면의 Firestore 진단 또는 기존 백업 기능으로 JSON 백업을 먼저 받아두세요. 삭제 후 복구는 백업 파일 없이는 어렵습니다.

구분경로설명

단일 문서boards/templates과목 원본 카드

단일 문서boards/rooms교실 데이터

분리 문서boards/_split/timetableMeta/main시간표 설정/교사조건/그룹

실행

1. 로그인 후 삭제 대상 확인을 먼저 누르세요.

2. 확인 문구에 전체초기화를 입력해야 삭제 버튼이 활성화됩니다.

초기화 후 기존 앱 화면은 반드시 새로고침하세요. 가능하면 Ctrl+F5로 강력 새로고침을 권장합니다.
